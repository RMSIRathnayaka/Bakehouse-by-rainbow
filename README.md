# Bakehouse by Rainbow

Bakehouse by Rainbow is a full-stack bakery management and online ordering system developed using React for the frontend and Python for the backend. The system allows customers to browse bakery products, customize cakes, place orders, and manage their shopping experience through a modern and responsive web application.

## Features

- User authentication system
- Product browsing and cake categories
- Custom cake ordering
- Shopping cart and order management
- Responsive frontend UI
- Backend API integration
- Database management
